# getFrame

Returns an object with `farcaster` and `openframes` keys, each containing an object with `frame` property extracting the frame metadata from the given htmlString, or containing `reports` property if the frame is invalid. Includes tags from the [Open Frames specification](https://github.com/open-frames/standard).

## Usage

```ts [example.ts]
import { getFrame, Frame } from "frames.js";

const { farcaster, openframes } = await getFrame({
  htmlString: "<html><head>...</head><body></body></html>",
  url: "https://framesjs-homeframe.vercel.app/",
});
console.log(farcaster, openframes);
/*
	{ 
		frame: {
			version: FrameVersion;
			postUrl: string;
			buttons?: FrameButtonsType | undefined;
			image: string;
			ogImage?: string | undefined;
			inputText?: string | undefined;
		}
	},
	{
		frame: {
			version: FrameVersion;
			postUrl: string;
			buttons?: FrameButtonsType | undefined;
			image: string;
			ogImage?: string | undefined;
			inputText?: string | undefined;
		}
	}
*/
```
